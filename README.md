# Pokemon Resell Tracker

Application pour tracker l'achat et la revente de cartes Pokemon.

## Installation

1. Extrait le dossier
2. Ouvre un terminal dans le dossier
3. Lance la commande :

\`\`\`bash
npm install
npm run dev
\`\`\`

4. Ouvre http://localhost:3000

## Fonctionnalités

- 📊 Dashboard avec statistiques
- 📦 Gestion des achats
- 💰 Gestion des ventes
- 📈 Calcul automatique des profits
- 💾 Stockage local des données

## Stack technologique

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
